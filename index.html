Process Configuration
Arrival: Time when the process enters the system (NEW → READY)
Burst: Total CPU execution time needed by the process
Priority: Lower number = higher priority (1 is highest)
Pages: Number of virtual memory pages allocated to the process
Remaining: CPU time still needed to complete execution
Turnaround Time (TAT): Completion Time - Arrival Time. Measures how long a process waits to finish after entering the system. Lower TAT means better responsiveness.
Throughput: Number of processes completed per time unit (processes/ms). Higher throughput = better system efficiency.
