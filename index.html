Instrucciones de uso
Guía paso a paso del simulador de Estados de Procesos
1
➕
Crear un proceso
Completa los campos del formulario y haz clic en + Agregar:

• PID — nombre único del proceso (ej. P1, Chrome, 101)
• Arrival — en qué unidad de tiempo llega al sistema (0–100 ms)
• Burst — cuánto tiempo de CPU necesita para terminar (1–50 ms)
• Priority — nivel de urgencia: 1 = alta prioridad, 10 = baja
• Páginas — memoria requerida en páginas (1–20)

El PID se incrementa automáticamente tras cada agregado.
2
🎮
Controles principales
• + Agregar — crea el proceso con los valores del formulario
• Aleatorio — genera un proceso con valores al azar para explorar rápido
• ↺ Reset — elimina todos los procesos y reinicia el reloj a T = 0
• 📂 TXT — carga múltiples procesos desde un archivo de texto
3
🔄
Cómo funciona la simulación
La simulación corre automáticamente cada segundo. Los procesos transicionan entre estados según su progreso:

NEW Recién creado — espera su tiempo de llegada
READY En cola — espera que el CPU quede libre
RUNNING Ejecutándose — el Core 0 lo está procesando
WAITING Bloqueado — esperando una operación de E/S
TERMINATED Terminado — completó su ejecución
4
🖥️
CPU Cores
Usa el selector CPU Cores (1, 2 o 4) para simular múltiples núcleos:

• Core 0 ★ — siempre ejecuta el proceso de mayor prioridad (menor número = mayor prioridad)
• Cores 1+ — visualizan los siguientes procesos en cola READY

La barra de progreso bajo cada core muestra el porcentaje completado del proceso actual.
5
📂
Cargar procesos desde archivo TXT
Presiona 📂 TXT (o 📄 Cargar TXT) y selecciona un archivo de texto. Cada línea define un proceso en formato CSV. Las líneas que comienzan con # son comentarios.

Formato: PID, Arrival, Burst, Priority, Páginas
# OS Visual Lab — ejemplo de procesos P1,0,5,2,3 P2,1,3,1,2 P3,2,8,3,4 P4,4,4,1,3 Usa ⬇ Ejemplo para descargar una plantilla lista para usar.
6
📤
Sincronizar con Scheduling
El botón 📤 Sync Scheduling exporta todos tus procesos actuales a la página de Scheduling. Desde allí podrás simular algoritmos de planificación como FCFS, SJF, Round Robin, Priority y más, usando los mismos procesos que definiste aquí.
7
📄
Exportar resultados a CSV
El botón 📄 Exportar CSV en la tabla inferior descarga un archivo con todos los procesos y su estado actual, incluyendo columnas: PID, Arrival, Burst, Remaining, Priority, Pages, State.
